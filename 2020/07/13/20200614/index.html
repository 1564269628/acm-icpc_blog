二分图统计dp：背包模型

ZOJ1462

$\textbf{algorithm}$
$\textbf{i) } \text{很容易想到这是一个二分图}$
$\quad \text{进行黑白二染色，如果非二分图，return false}$
$\quad \text{如果是二分图，考虑 dp 计数}$

$\textbf{ii) 进行黑白二染色之后，每个连通分量的队伍安排，是确定的}$
$\quad \text{ 黑的只能去黑的组，白的只能去白的组}$

$\quad \quad \textbf{1. 很显然根据连通分量划分 dp 阶段}$
$\quad \quad \quad \textbf{for } \forall i \in [1, cc], team(i, 0), team(i, 1) $
$\quad \quad \quad \text{分别表示连通分量 } i \text{ 中两种染色的点有多少}$
$\quad \quad \textbf{2. 可以计算出每个连通分量两个组人员的差}$
$\quad \quad \quad \textbf{diff}(i) = team(i, 0) - team(i, 1)$

$d(i, j) \text{ 表示第 } i \text{ 个连通分量两组人的差异是 }j$

$\text{这里 } d(i, j) \text{ 表示这个状态是否存在}$
$\text{start: } d(0, 0) = 1$
$\text{因为有负值存在，这里我们算的时候，都加上一个 } n \text{ 做调整}$
$j \to j+n$

$\textbf{iii) 如何输出？}$
$dp(0 \rightarrow cc-1), \text{如果能够存在某个 diff 差异} = ans$
$d(cc, ans) = 1$

$\textbf{for } ans = [0 \cdots n]$
$\quad \textbf{if } d(cc, ans) = 1, \text{print}(ans), \textbf{ return}$

$\textbf{print}(ans):$
$\textbf{for } \forall i = cc-1 \text{ downto } 0$
$\quad \textbf{if }d(i, ans-\text{diff}(i))=1, ans-\text{diff}(i), [team0]$
$\quad \textbf{if }d(i, ans+\text{diff}(i))=1, ans+\text{diff}(i), [team1]$
$[team0], [team1] \text{ is the answer}$

总结，本例中的图本来就是一个二分图
$d(i+1, j+\text{diff}(i))=1$ 的连通分量，其中的点属于 team1，那么
$d(i+1, j-\text{diff}(i))=1$ 的连通分量，其中的点一定属于 team2
$\text{diff}$ 表示每个连通分量之间 不同染色点之间的差异
